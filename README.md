## Install
Clone the repo and install dependencies:

```bash
git clone https://github.com/nzrylmz/electron-react-boilerplate-micro.git your-project-name
cd your-project-name
npm install
```

## Run
```bash
npm run electron:serve
```

## Build
```bash
npm run electron:build
```
